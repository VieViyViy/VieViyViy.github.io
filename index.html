üîç Abstract

This study explores deep learning models‚Äî1D CNN, LSTM, and GRU‚Äîfor classifying five types of breathing patterns (Hold, Deep, Deep Quick, Quick, Normal) using time-series sensor data. Each architecture is trained and evaluated using accuracy, F1-score, and confusion matrices. 1D CNN achieved highest accuracy (96%) but showed overfitting. GRU balanced speed and performance, suggesting its practicality for real-time health applications.

üß™ Methodology

  • Data Preprocessing: Zero values replaced and removed; normalized and downsampled to equal sequence lengths.
  • Label Encoding & Split: Categorical mapping; stratified 70-15-15 train-validation-test split.
  • Model Architectures:
    • üü¶ 1D CNN: Two convolutional layers ‚Üí MaxPooling ‚Üí Dropout ‚Üí Dense ‚Üí Softmax.
    • üü® LSTM: Two stacked LSTM layers ‚Üí LayerNorm ‚Üí Dropout ‚Üí Dense ‚Üí Softmax.
    • üü• GRU: Two stacked Bidirectional GRUs ‚Üí LayerNorm ‚Üí Dropout ‚Üí Dense ‚Üí Softmax.
  • Training: Adam optimizer (LR=1e-4), L2 regularization, dropout, early stopping.

üìä Results

1D CNN: Highest accuracy (96%) but overfitting.
LSTM: Stable performance (90%) with long-term pattern learning.
GRU: Balanced model (91%), fastest to converge, good generalization.

üßæ Conclusion

While 1D CNN produced the best accuracy, it required aggressive regularization. LSTM handled temporal patterns well, but GRU offered a strong trade-off between speed, performance, and generalization. GRU is recommended for real-time applications. Future improvements may involve attention mechanisms or hybrid architectures.
